### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.72
- Хуки agent-stop, которые бесконечно блокируют выполнение, теперь завершают ход после восьми последовательных блокировок и получают флаг `stop_hook_active` для самостоятельного ограничения.
- Многоходовые субагенты включены всегда, поиск инструментов доступен для Claude Haiku 4.5 и новее, а запланированные промпты могут направлять занятого агента.
- Управление плагинами теперь поддерживает обновление, включение, отключение и удаление плагинов, MCP-серверов и skills, включая установку skills из файлов, URL или каталогов.
- `/model --session` позволяет менять модель, уровень рассуждений или размер контекстного окна только для текущей сессии, не изменяя глобальные настройки.

### GitHub Copilot CLI - 1.0.73
- Субагенты Anthropic продолжают работать при настройке дополнительных каталогов.
- Относительные ссылки в инструкциях custom agent теперь разрешаются относительно расположения файла агента.

### GitHub Copilot CLI - 1.0.74
- Добавлена поддержка манифестов Open Plugin Spec v1 и конфигурации `mcp.json`.
- Интеграции с IDE надёжно переподключаются, когда CLI перезагружает MCP-серверы или меняет каталог, а временные шкалы субагентов сохраняют порядок промптов и показывают, пришёл ли промпт от основного агента или субагента.
- Добавлена поддержка модели `gemini-3.6-flash` и приветственный экран для выбора песочницы по умолчанию при первом запуске.
- Skills теперь учитывают `disable-model-invocation`, а мастера `/mcp add` и `/mcp edit` сохраняют символы `=` в значениях переменных окружения.
- Режим планирования может использовать артефакты в каталоге сессии и собственную модель, выбранную через `/model plan`, продолжая блокировать изменения рабочей области.

### Visual Studio Code - 1.130
- В окне Agents diff теперь показывается актуальное число добавленных и удалённых строк для каждого файла, многофайловый просмотр стал компактнее, а изоляция через worktree доступна для всех harness агентов.
- Assisted permissions позволяют языковой модели оценивать риск вызова инструмента во время задач агента и сокращают количество повторных запросов подтверждения.
- Пользователи Copilot Business и Enterprise теперь видят суммарное использование AI-кредитов за текущий расчётный период в меню состояния Copilot.

### Visual Studio 2022 - 17.14.37
- Исправлена ошибка запроса Copilot, из-за которой при значении `max_output_token` меньше 16 возвращалась внутренняя ошибка.

### Visual Studio 2026 - Июльское обновление 18.8.0
- В Visual Studio появились встроенные skills .NET и Azure для настройки рабочих процессов Copilot-агента; их можно просматривать и включать в окне выбора инструментов.
- Окно использования Copilot теперь показывает метрики в реальном времени и заранее предупреждает о приближении или достижении лимитов.
- Новый вариант Agent (предварительная версия) на базе Copilot SDK рассчитан на более короткие ответы и лучшее выполнение задач с первой попытки.
- Выбранный код теперь можно проверять с помощью Copilot, получая внутристрочные комментарии и предложения прямо в редакторе.
- Владельцы организаций могут задавать общие инструкции для Copilot, а ветви можно прикреплять к Copilot Chat как контекст.
