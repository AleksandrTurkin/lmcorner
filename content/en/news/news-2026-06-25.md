### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.64
- The CLI now shows the pay-as-you-go additional usage budget at launch, refreshes it after spend-limit rejections, and displays a friendlier message when the additional usage limit is reached.
- BYOK OpenAI-compatible providers now support WebSocket responses.
- You can configure the Rubber Duck subagent in `/subagents`, including a complementary model strategy that picks an opposite-family model.
- Autopilot now returns to interactive mode after the agent calls `task_complete`, so the next prompt does not stay in autopilot unexpectedly.

### GitHub Copilot CLI - 1.0.65
- `/cd` now persists the working directory so resumed sessions return to it, and custom agents are rediscovered after changing directories.
- Added the `copilot skill` subcommand and `/skill` alias to list, add, and remove skills from a file, URL, or directory.
- Custom-agent subagent model selections are now preserved when using BYOK providers.
- `userPromptSubmitted` hook `additionalContext` is now included in the model-facing prompt.

### Visual Studio Code - 1.125
- The Copilot status dashboard now shows how much of your additional spend budget has been consumed, helping you stay ahead of overage charges.
- Agents now handle forwarded ports better in remote workspaces by rewriting the URL when a forwarded port changes and notifying the agent.
- Managed GitHub Copilot settings can now be delivered through native MDM on Windows and macOS, so organizations can enforce Copilot policies through existing device-management tooling.

### Visual Studio Code - 1.126
- You can now see the total cost of an entire chat session, making it easier to spot expensive Copilot conversations.
- The model customization UI now combines context size and reasoning effort into a single picker.
- A single agent host Copilot session can now hold multiple chats side by side, with each chat persisted and restored across reloads.
- Code comments created during reviews can now be stored on the agent host, letting review skills and agents work with server-side comments and pull request review items.

### Visual Studio 2022 - 17.14.35
- Improved messaging around additional Copilot usage and plan limits under usage-based billing.

### Visual Studio 2026 - 18.7.2
- Fixed an issue where Copilot Chat could fail to open because of a `RemoteInvocationException`.
