### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.66
- Сессии сабагентов теперь наследуют ограничения на инструменты из родительской сессии.
- CLI теперь явно показывает предупреждения и ошибки, если не удалось загрузить кастомных агентов на host-стороне.
- Добавлена поддержка Claude Sonnet 5.
- Вызовы инструментов теперь могут продолжаться после таймаута хуков.
- MCP OAuth для серверов Microsoft Entra с vanity-доменами теперь корректно обновляет токены и повторно проходит аутентификацию.

### GitHub Copilot CLI - 1.0.67
- Добавлена поддержка Claude Opus 4.8 Fast, а Claude Opus 4.6 Fast переведен в deprecated.
- Copilot теперь умеет читать вывод и останавливать отсоединенные фоновые shell-команды.
- Таймлайн теперь использует компактный формат highlight reel с однострочными строками для инструментов и reasoning.
- В relay-сессиях появились автодополнения `@` для файлов и `#` для GitHub-ссылок.
- Логи LSP-серверов теперь можно просматривать через `/lsp logs` и `read_agent`.
- `/pr auto` теперь может продолжать работу через CI, review и merge queue.
- В `/settings` теперь можно настраивать лимиты параллелизма и глубины для сабагентов.

### Visual Studio Code - 1.127
- Добавлена поддержка `/troubleshoot` для сессий Agent Host Copilot CLI.
- Инструменты агента для встроенного браузера вышли из experimental.
- Agent Host теперь показывает, какие файлы изменились во время сессии.
- В agent-host-сессиях Claude-агент теперь может работать через подписку Anthropic вместо квоты Copilot.
- Если long context не требует доплаты, VS Code теперь автоматически использует его вместо меньшего default-tier режима.
- В Agent Customizations теперь можно настраивать встроенные инструменты и инструменты расширений для Agent Host.

### Visual Studio 2026 - 18.7.3
- Исправлена ошибка `VSCopilot ChatError`, из-за которой Visual Studio не могла найти сервис `Microsoft.VisualStudio.Copilot.ICopilotInteractionService`.
- Исправлен `NullReferenceException`, который мог помешать инициализации окна Copilot Chat.
