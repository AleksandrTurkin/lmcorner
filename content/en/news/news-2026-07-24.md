### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.72
- Agent-stop hooks that block indefinitely now end the turn after eight consecutive blocks and receive a `stop_hook_active` flag to self-limit.
- Multi-turn subagents are always enabled, tool search is available for Claude Haiku 4.5 and later, and scheduled prompts can steer a busy agent.
- Plugin management now supports updating, enabling, disabling, and removing plugins, MCP servers, and skills, including installing skills from files, URLs, or directories.
- `/model --session` can change the model, reasoning effort, or context window for the current session without changing global settings.

### GitHub Copilot CLI - 1.0.73
- Anthropic subagents continue working when additional directories are configured.
- Relative links in custom agent instructions are now resolved from the agent file's location.

### GitHub Copilot CLI - 1.0.74
- Added support for Open Plugin Spec v1 manifests and `mcp.json` configuration.
- IDE integrations reconnect reliably when the CLI reloads MCP servers or changes directory, while subagent timelines preserve prompt order and identify whether prompts came from the main agent or a subagent.
- Added support for the `gemini-3.6-flash` model and a first-run splash for opting into the default sandbox.
- Skills now honor `disable-model-invocation`, and `/mcp add` and `/mcp edit` preserve `=` characters in environment-variable values.
- Plan mode can use session-folder planning artifacts and has its own selectable model through `/model plan`, while still blocking workspace mutations.

### Visual Studio Code - 1.130
- Agents window diffs now show live file-level insertion and deletion counts, use a more compact multi-file layout, and support worktree isolation for every agent harness.
- Assisted permissions let the language model evaluate tool-call risk during agent tasks to reduce repeated approval prompts.
- Copilot Business and Enterprise users can now see aggregate AI credit usage for the current billing cycle in the Copilot status menu.

### Visual Studio 2022 - 17.14.37
- Fixed a Copilot query failure that returned an internal error when `max_output_token` was less than 16.

### Visual Studio 2026 - July Update 18.8.0
- Visual Studio now includes built-in .NET and Azure skills for customizing Copilot agent workflows; they can be reviewed and enabled from the tool picker.
- The Copilot Usage window now provides real-time usage metrics and proactive alerts as limits are approached or reached.
- A new Agent (Preview), powered by the Copilot SDK, aims for shorter responses and better first-attempt task completion.
- Selected code can be reviewed with Copilot, which adds inline comments and suggestions directly in the editor.
- Organization owners can apply custom instructions to Copilot, and branches can be attached to Copilot Chat as context.
