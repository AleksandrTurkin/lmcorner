### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.68
- Добавлена поддержка модели kimi-k2.7-code.
- На экране агентов теперь можно просматривать, возобновлять и переключать сессии.
- Инструменты IDE теперь остаются доступными при кратковременных разрывах соединения с IDE и автоматически восстанавливаются после переподключения.
- Кастомные агенты теперь сохраняют свои ограничения на инструменты и во вложенных сабагентах.

### Visual Studio Code - 1.128
- Сессии Claude-агента теперь могут содержать несколько связанных чатов, чтобы можно было сравнивать подходы, ветвиться от предыдущих ходов и выполнять работу параллельно.
- В окне Agents теперь можно запускать быстрые чаты без предварительного выбора workspace.
- Read-only чаты сабагентов позволяют в реальном времени следить за делегированной работой в окне Agents, не прерывая исполнителя.
- Copilot Vision теперь доступен в general availability: в Chat можно прикреплять изображения и PDF, а агент может читать изображения через вызовы инструментов.
- В agent-host-сессиях с BYOK теперь можно использовать собственные модели и настраивать utility model по умолчанию для фоновых задач вроде генерации заголовка чата и сообщения коммита.
