### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.62
- You can now configure a subagent's model, reasoning effort, and context tier from settings or the `/subagents` and `/agents` pickers.
- `/every` and `/after` can now schedule slash commands, making recurring agent workflows possible.
- The `/diff` experience gained content search, match highlighting, `n`/`N` navigation, a file tree sidebar, and an inline comment editor.
- The MCP server configuration form now uses a simpler picker-based flow.

### GitHub Copilot CLI - 1.0.63
- Blocked image attachments now explain how to enable vision features, switch to a vision-capable model, or try a different image instead of failing with a vague error.
- Added the `deferTools` MCP server option so tools can stay available even when tool search is enabled.
- Agent mode is now tracked per session, so it no longer carries over when you create, clear, or switch sessions.
- Experimental `/rewind` no longer requires git and can restore only the files Copilot changed, with a choice to rewind just the conversation or both conversation and files.
