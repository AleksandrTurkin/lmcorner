### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.42
- Улучшены предупреждения об ошибках MCP с готовыми командами диагностики и выводом stderr для упрощения отладки.
- Добавлен флаг `-C` для смены рабочей директории перед запуском, а также улучшен экспорт удалённых сессий для репозиториев вне GitHub и директорий без репозитория.
- Добавлен экспериментальный агент rubber-duck для GPT-сеансов и исправлены ошибки извлечения пакетов на Windows во время обновления.

### GitHub Copilot CLI - 1.0.43
- Добавлен переключатель имени пользователя в `/statusline` и улучшен Auto mode с серверной маршрутизацией моделей.
- Улучшены подсказки возобновления и обработка сессий при нескольких активных сессиях.
- Исправлена потенциальная RCE-уязвимость во вложенных bare-репозиториях и улучшено завершение дочерних процессов MCP.

### GitHub Copilot CLI - 1.0.44
- Исправлено автодополнение путей для `/add-dir`, добавлена поддержка слэш-команд в середине ввода и улучшена обработка shell-команд и alias.
- Добавлены хуки `userPromptSubmitted`, которые могут обрабатывать запросы напрямую без обращения к модели.
- Улучшены производительность и UX: более быстрое переключение аккаунтов, исправления отображения квоты и устойчивость к сбоям на некоторых версиях Node.

### Visual Studio Code - 1.119
- Дата выпуска: 6 мая 2026 — улучшено взаимодействие агентов с браузером, добавлен OpenTelemetry для трассировки сессий агентов, оптимизировано использование токенов, улучшена безопасность и превью Markdown.
- Влияние: расширяет интеграцию агентов (поделиться вкладками браузера и трассировка) и повышает наблюдаемость и доверие при разработке.
- Источник: https://code.visualstudio.com/updates/v1_119
