### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.69
- Added CLI OAuth authentication for MCP servers and `/mcp list` to inspect attached servers and their status while the agent is working.
- MCP servers can now be enabled or disabled mid-turn, and OAuth-gated servers re-register their tools after reconnecting.
- The model picker is easier to browse and filter, while the CLI warns when static context is consuming most of the prompt budget.

### GitHub Copilot CLI - 1.0.70
- Added support for the GPT-5.6 model.
- Repository settings can extend URL, MCP, and skill deny lists, and `/settings` and `/model` accept `--repo` and `--local` scopes.
- Background-agent notifications are no longer duplicated after a blocking `read_agent` returns its result.

### GitHub Copilot CLI - 1.0.71
- Autopilot prompt runs no longer hang when a background shell or agent outlives the turn and now honor the configured task-wait timeout.
- The `/subagents` model picker preserves each agent's reasoning effort and context tier.
- MCP tool lists stay current as servers change, and plan mode blocks workspace-mutating built-in calls.

### Visual Studio Code - 1.129
- Agent sessions can now run in a dedicated agent-host process and connect to multiple VS Code windows.
- The Agents window adds an experimental docked editor panel for reviewing agent-generated files and diffs.
- Chat prompts can run terminal commands directly with the `!` prefix.
- Agent Host sessions gain tools to create, observe, and steer other sessions and chats, while new-session defaults and worktree isolation are easier to reuse.
- The Copilot agent harness in the Agents window now supports BYOK models, and Copilot authentication works with GitHub Enterprise instances.

### Visual Studio 2022 - 17.14.36
- Updated GitHub Copilot to support Azure MCP 2.0.4.

### Visual Studio 2026 - July Update 18.7.4
- Visual Studio now includes built-in .NET and Azure skills for customizing Copilot agent workflows.
- The refreshed Copilot Usage window provides real-time usage metrics and proactive alerts near limits.
- A new Agent (Preview) option uses the Copilot SDK for shorter responses and better first-attempt task completion, and selected code can receive inline Copilot review comments.
- Organization owners can apply custom Copilot instructions, and branches can be attached to Copilot Chat as context.
