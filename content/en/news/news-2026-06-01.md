### GitHub Copilot CLI - 1.0.56
- Free and Student users can now choose models other than Auto in the model picker.
- The model picker now shows the correct total context window size for each pricing tier.
- Added the `builtInAgents.rubberDuck` setting so the rubber duck agent can be enabled or disabled through `copilot config`.
- BYOK provider configuration now applies correctly to ACP sessions.

- Context window tier selection now persists across session events and resume flows, so request, compaction, and truncation limits stay aligned with the selected tier.
- The `web_fetch` tool now prefers Markdown when the source supports content negotiation, producing cleaner documentation context for the agent.
