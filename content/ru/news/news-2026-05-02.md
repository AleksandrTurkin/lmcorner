### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.36
- Команда `/keep-alive` стала доступна без experimental mode, чтобы Copilot CLI не давал системе уходить в сон во время работы.
- Добавлены `/remote on` и `/remote off`, а также более понятный вывод статуса remote-подключения.
- Отключённые skills больше не показываются в списке slash-команд.
- Улучшена поддержка пользовательских instruction files в `.gitignored` директориях.

### GitHub Copilot CLI - 1.0.37
- Добавлена команда `copilot completion <bash|zsh|fish>` для генерации shell completion scripts.
- Улучшен skill picker: элементы с ошибками и предупреждениями теперь остаются полностью видимыми.
- Ответы `/ask` теперь корректно рендерят Markdown-таблицы и форматированные ссылки.

### GitHub Copilot CLI - 1.0.39
- Для ACP sessions добавлены slash-команды `/compact`, `/context`, `/usage` и `/env`.
- ACP clients теперь могут переключать allow-all permission mode через конфигурацию сессии.
- Добавлено быстрое перемещение текущей задачи или shell-команды в фон.
- Улучшен вывод `/remote`: теперь он показывает более полезные подсказки по состоянию подключения.

### GitHub Copilot CLI - 1.0.40
- Добавлена headless-поддержка OAuth `client_credentials` для MCP servers.
- Команда `/research` получила orchestrator/subagent model для более надёжного deep research.
- История сессий, file tracking и команда `/chronicle` стали доступны всем пользователям.
- Skills стали доступны как slash-команды в ACP clients, а сами ACP clients теперь могут показывать и переключать custom agents.
- Добавлены показ live plan в ACP clients и более плавный streaming ответов ассистента.

### Visual Studio Code - 1.118
- Semantic indexing для не-GitHub репозиториев стал доступен всем пользователям, поэтому Copilot лучше понимает рабочие области вне GitHub и ADO.
- Добавлен агентный инструмент `githubTextSearch`, чтобы Copilot мог выполнять точный текстовый поиск по GitHub-репозиторию или всей организации.
- Для skills добавлен выделенный subagent context, чтобы крупные skill-запуски не перегружали основной чатовый контекст.
- Добавлено расширение Chat Customizations Evaluations для анализа prompt files, custom agents, instructions и skills.

### Visual Studio 2022 - 17.14.29
- Исправлено зависание Copilot в agent mode при использовании многострочных terminal-команд для редактирования файлов или доступа к ним.
- Исправлены зависания из-за многострочных terminal-команд, сгенерированных моделью, когда в PowerShell был включён Posh.

### Visual Studio 2022 - 17.14.30
- Исправлено зависание Copilot sessions в состоянии ожидания разрешений.
- Исправлены сбои Copilot при работе с моделью Opus 4.6.
- Исправлены ошибки Copilot sessions после возобновления приостановленной сессии.
- Добавлена поддержка GitHub Copilot через Azure extension workload.

### Visual Studio 2022 - 17.14.31
- Исправлена проблема в расширении GitHub Copilot for Azure, из-за которой не работал инструмент Query Azure Resource Graph.

### Visual Studio 2026 - 18.5.1
