### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.66
- Subagent sessions now keep the parent session's tool restrictions.
- The CLI now surfaces warnings and errors when host custom agents fail to load.
- Added support for Claude Sonnet 5.
- Tool calls can now continue when hooks time out.
- MCP OAuth against Microsoft Entra vanity domains now refreshes and re-authenticates correctly.

### GitHub Copilot CLI - 1.0.67
- Added support for Claude Opus 4.8 Fast and deprecated Claude Opus 4.6 Fast.
- Copilot can now read output from and stop detached background shell commands.
- The timeline now uses a compact highlight-reel view with single-line tool and reasoning rows.
- Relay sessions now support `@` file and `#` GitHub reference completions.
- You can now view LSP server logs from `/lsp logs` and `read_agent`.
- `/pr auto` can now keep working through CI, review, and merge queue.
- `/settings` can now configure subagent concurrency and depth limits.

### Visual Studio Code - 1.127
- Added `/troubleshoot` support for Agent Host Copilot CLI sessions.
- Integrated browser agent tools are now out of experimental.
- Agent Host can now show which files changed during a session.
- The Claude agent can now run through an Anthropic subscription instead of Copilot quota in agent-host sessions.
- When long context has no extra cost, VS Code now automatically uses it instead of offering a smaller default tier.
- Agent Customizations can now configure built-in and extension tools for Agent Host.

### Visual Studio 2026 - 18.7.3
- Fixed a `VSCopilot ChatError` where Visual Studio could not find the `Microsoft.VisualStudio.Copilot.ICopilotInteractionService` service.
- Fixed a `NullReferenceException` that could prevent the Copilot Chat window from initializing.
