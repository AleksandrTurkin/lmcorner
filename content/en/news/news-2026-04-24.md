### GitHub Copilot CLI - 1.0.35
- Added tab completion for slash commands, including arguments and subcommands.
- Improved session management with named sessions, deletion commands, smarter `--continue` behavior, and a richer session picker with branch and status details.
- Expanded agent workflows with self-correcting custom tool calls, HTTP hook support, better plugin and LSP behavior, and Windows shell access for the configure agent flow.

### Visual Studio Code - 1.117
- Added BYOK for Copilot Business and Enterprise so teams can connect their own model API keys directly in VS Code chat.
- Improved chat and agent workflows with incremental rendering, agent session sorting, and system notifications for background terminal commands.
- Expanded agent tooling with the evolving Visual Studio Code Agents app and better terminal support for launching Copilot CLI and showing agent CLI titles.

### Visual Studio 2022 - April Update 17.14
- Copilot Agent Mode moved out of preview and gained clearer progress indicators, smarter planning, optional manual build control, and reliability fixes.
- Added GPT-5 support in Visual Studio chat.
- Made MCP support generally available in Copilot, with broader OAuth support, easier server setup, and governance controls.
- Added bring-your-own-model support in chat plus more precise Copilot completions through partial accept and hidden-by-default next edit suggestions.

### Visual Studio 2026 - April Update 18.5
- Copilot agents can now automatically discover and use skills defined in a repository or user profile.
- Visual Studio can start new cloud agent sessions backed by Copilot coding agent directly from chat.
- Added custom agents, customizable Copilot shortcut keys, IntelliSense priority over Copilot completions, and a dedicated chat history panel.
- Introduced Copilot-powered text auto-decoding and a debugger agent workflow for issue-to-resolution debugging.
