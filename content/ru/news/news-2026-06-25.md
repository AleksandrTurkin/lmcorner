### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.64
- CLI теперь показывает бюджет дополнительного pay-as-you-go usage при запуске, обновляет его после отклонения запроса из-за лимита расходов и выводит более понятное сообщение при достижении лимита дополнительного использования.
- BYOK-провайдеры, совместимые с OpenAI, теперь поддерживают ответы по WebSocket.
- В `/subagents` теперь можно настраивать сабагента Rubber Duck, включая стратегию дополнительной модели из другой модельной семьи.
- Режим Autopilot теперь автоматически возвращается в интерактивный режим после вызова агентом `task_complete`, чтобы следующий запрос не оставался в Autopilot.

### GitHub Copilot CLI - 1.0.65
- Команда `/cd` теперь сохраняет рабочий каталог, поэтому при возобновлении сессии CLI возвращается в него, а после смены каталога заново обнаруживает кастомных агентов.
- Добавлены подкоманда `copilot skill` и алиас `/skill` для просмотра, добавления и удаления skills из файла, URL или каталога.
- При использовании BYOK теперь сохраняется выбор модели сабагента у кастомных агентов.
- Дополнительный контекст `additionalContext` из хука `userPromptSubmitted` теперь передается в prompt, который видит модель.

### Visual Studio Code - 1.125
- Панель состояния Copilot теперь показывает, какая доля дополнительного бюджета уже израсходована, чтобы было проще контролировать перерасход.
- В удаленных рабочих средах агенты теперь корректнее работают с проброшенными портами: VS Code переписывает URL при изменении такого порта и сообщает об этом агенту.
- Управляемые настройки GitHub Copilot теперь можно доставлять через нативный MDM в Windows и macOS, чтобы организации могли применять политики Copilot через существующие средства управления устройствами.

### Visual Studio Code - 1.126
- Теперь можно видеть полную стоимость всей чат-сессии, чтобы проще находить дорогие диалоги Copilot.
- В настройке модели размер контекста и уровень reasoning effort объединены в один picker.
- В одной agent host Copilot-сессии теперь можно держать несколько чатов рядом, причем каждый чат сохраняется и восстанавливается после перезагрузки.
- Комментарии к коду, созданные во время review, теперь могут храниться на стороне agent host, чтобы review-skills и агенты работали с серверными комментариями и замечаниями к pull request.

### Visual Studio 2022 - 17.14.35
- Улучшено описание дополнительного использования Copilot и ограничений плана при usage-based billing.

### Visual Studio 2026 - 18.7.2
- Исправлена проблема, из-за которой Copilot Chat мог не открываться из-за исключения `RemoteInvocationException`.
