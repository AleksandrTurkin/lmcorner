### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.68
- Added support for the kimi-k2.7-code model.
- The agents screen can now browse, resume, and switch between sessions.
- IDE tools now stay available through transient IDE disconnects and recover automatically after reconnection.
- Custom agents now preserve their tool filters in nested subagents.

### Visual Studio Code - 1.128
- Claude agent sessions can now hold multiple related chats in one session, so you can compare approaches, branch from earlier turns, and run work in parallel.
- The Agents window now supports quick chats without selecting a workspace first.
- Read-only subagent chats let you follow delegated work live from the Agents window without interrupting the worker.
- Copilot Vision is now generally available, with image and PDF attachments in Chat and agent tool-call access to images.
- Agent-host BYOK sessions can now use BYOK models and configure a default utility model for background tasks such as chat title and commit message generation.
