### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.57
- Во всех репозиториях только с Azure DevOps встроенный GitHub MCP-сервер теперь предоставляет инструмент `web_search` вместо полного отключения.
- Ошибки хука `preToolUse` теперь блокируют вызов инструмента, а не пропускают его молча, а настройки таймаута MCP-серверов больше не теряются при изменении списка инструментов.
- Улучшены восстановление сеансов и обработка запросов: корректнее работает resume после сбоев, `/skills` лучше обрабатывает пути в кавычках, а ошибки лимита GitHub API в диагностике аутентификации стали понятнее.

### GitHub Copilot CLI - 1.0.58
- Rubber Duck теперь включен по умолчанию.
- Remote JSON RPC теперь включен по умолчанию.
- В `/experimental` появились отложенные запросы через `/every` и `/after`, а также новый GitHub-интерфейс для issues, pull requests и gists.

### GitHub Copilot CLI - 1.0.59
- Добавлена команда `/voice` для диктовки запросов с помощью локальных speech-to-text моделей.

### GitHub Copilot CLI - 1.0.60
- Добавлен максимальный уровень reasoning effort для моделей Anthropic, и теперь все уровни доступны на любом плане.
- Появился параметр `builtInAgents.rubberDuckAutoInvoke` для управления автоматическим вызовом rubber-duck-агента.
- Инструменты MCP, добавленные или удаленные сервером посреди хода, теперь становятся доступными сразу, а вложения BYOK размером больше 5 MiB корректно отправляются через OpenAI Responses.
- Инструкции кастомных агентов больше не дублируются на каждом ходе, что уменьшает расход контекстного окна.

### GitHub Copilot CLI - 1.0.61
- Обновлены окно выбора `/agents` и мастер Create New Agent.
- MCP-серверы теперь автоматически подхватываются из `.github/mcp.json`, а `/after` и `/every` появились в списке experimental slash-команд.
- Добавлена поддержка Claude Fable 5 и улучшена совместимость Gemini с MCP-инструментами, использующими nullable-схемы.
- Для телеметрии добавлены поддержка mTLS/частных CA и стандартный экспорт через OTLP HTTP/protobuf.


### Visual Studio Code - 1.123
- Сеансы чата теперь могут автоматически синхронизироваться с вашей учетной записью GitHub, а `/chronicle` добавляет поиск по истории работы, стендапы и персональные подсказки по продуктивности.
- В preview-версии окна Agents теперь можно держать открытыми несколько сеансов рядом и сравнивать или проверять работу агентов параллельно.
- В Copilot CLI Insiders появился preview-агент research для глубоких исследований с цитируемым Markdown-отчетом.
- Совместимые модели Anthropic и OpenAI теперь поддерживают контекстное окно до 1 млн токенов.

### Visual Studio Code - 1.124
- Autopilot теперь включен по умолчанию, а Advanced Autopilot может использовать небольшую служебную модель, чтобы решать, действительно ли агент уже завершил задачу.
- Фоновая отправка позволяет запускать новый агентный сеанс через Alt+Enter и сразу переходить к следующему запросу.
- Навигация по сеансам с клавиатуры и автоматическое восстановление раскладки ускоряют работу с большим числом агентных сеансов и их возврат после перезагрузки.
- История встроенного браузера и новый режим `typeInPage` с параметром submit уменьшают число проходов в browser-driven сценариях агента, а политики плагинов теперь используют тот же Copilot policy file, что и Copilot CLI.

### Visual Studio 2022 - 17.14.34
- Режим Copilot Agent Mode вышел из preview в Visual Studio 2022 и теперь может использовать контекст через MCP из GitHub, CI-конвейеров и систем мониторинга.
- Последние улучшения Agent Mode добавляют более понятные индикаторы прогресса, кэширование запросов, очистку working set, живой Markdown-инструмент для планирования и ручное управление автоматическими сборками.
- В Visual Studio 2022 появились поддержка GPT-5 в Copilot Chat и общедоступная поддержка MCP, включая более широкий OAuth-сценарий и более простой способ подключения MCP-серверов.
- Теперь к чату можно подключать собственные модели Anthropic, Google и OpenAI, частично принимать code completions и скрывать next edit suggestions до момента, когда вы сами захотите их открыть.
- В Copilot Chat также появились модели Gemini 2.5 Pro, GPT-4.1, Claude Sonnet 4 и Claude Opus 4.

### Visual Studio 2026 - Июньское обновление 18.7.0
- Окно использования Copilot теперь показывает метрики usage-based billing в реальном времени, упреждающие предупреждения о приближении к лимитам и настраиваемый порог уведомлений.
- Предложения для следующего редактирования на большом расстоянии позволяют GitHub Copilot предлагать связанные правки в любом месте текущего файла, а не только рядом с курсором.
- Проверка доверия для MCP-серверов теперь отслеживает и конфигурацию сервера, и загруженные ресурсы, запрашивая повторное подтверждение при изменениях.
- Pull request теперь можно добавлять напрямую в Copilot Chat, а сценарии обновления MSVC в GitHub Copilot modernization agent для C++ стали общедоступными.
