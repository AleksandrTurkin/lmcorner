### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.45
- Добавлена команда `/autopilot` для переключения между интерактивным и автопилотным режимами.
- OpenTelemetry теперь соответствует семантике GenAI: добавлена метрика gen_ai.client.operation.duration для времени выполнения инструментов.

### GitHub Copilot CLI - 1.0.46
- Появляется предупреждение, если версия CLI устарела и может быть потерян доступ к премиум-моделям.

### GitHub Copilot CLI - 1.0.47
- Подписчики Copilot Max теперь видят корректные модели, доступные их тарифу.
- `--resume` поддерживает сессии облачных агентов Copilot, где агент ещё не запушил изменения.

### GitHub Copilot CLI - 1.0.48
- В селекторе моделей отображаются реальные цены токенов для тарифных пользователей.
- `/context` показывает корректные лимиты токенов для всех моделей.
- Клиенты ACP получают обновлённые параметры конфигурации при смене активной модели.
- Диалог `/ask` больше не предлагает дополнительные ответы, которые он не может получить.
- Содержимое навыков, внедряемое в модель, больше не включает YAML frontmatter.

### Visual Studio Code - 1.120
- Окно Agents стало доступно в Stable для оркестрации задач агентов между проектами.
- Улучшения BYOK: корректное отображение использования токенов для BYOK-моделей и настройка уровня мышления (thinking effort).
- Оценка риска для терминальных команд теперь показывает сгенерированное ИИ объяснение и бейдж уровня риска.
- Сжатие вывода терминала уменьшает использование контекста при отправке больших выводов в модель.

### Visual Studio 2026 - 18.6.0
- Просмотр и управление навыками агента (Agent Skills) из окна чата Copilot: панель навыков для просмотра, редактирования и поиска навыков.
- Сводный дифф по нескольким файлам для изменений Copilot: просмотр и принятие/отмена правок по нескольким файлам в одном окне.
- Индикатор окна контекста показывает точное использование контекста и кнопку «Сжать» для освобождения места.
- Режим планирования (Planning mode) для составления плана реализации до внесения изменений, сохраняемый в markdown.
