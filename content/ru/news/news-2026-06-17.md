### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.62
- Теперь для сабагента можно настраивать модель, уровень reasoning effort и размер контекстного окна через настройки или пикеры `/subagents` и `/agents`.
- Команды `/every` и `/after` теперь умеют запускать slash-команды по расписанию, что упрощает регулярные агентные сценарии.
- В режиме `/diff` появились поиск по содержимому, подсветка совпадений, навигация через `n`/`N`, дерево файлов и встроенный редактор комментариев.
- Форма настройки MCP-сервера стала удобнее благодаря новому flow с picker-интерфейсом.

### GitHub Copilot CLI - 1.0.63
- Если вложение с изображением заблокировано, CLI теперь подсказывает, как включить vision-возможности, выбрать совместимую модель или попробовать другое изображение, вместо неясной ошибки.
- Добавлен параметр MCP-сервера `deferTools`, который позволяет держать инструменты доступными даже при включенном tool search.
- Режим агента теперь отслеживается на уровне отдельной сессии и больше не переносится при создании, очистке или переключении сессий.
- Experimental-команда `/rewind` больше не требует git и может откатывать только файлы, измененные Copilot, с выбором между откатом только диалога или диалога вместе с файлами.
