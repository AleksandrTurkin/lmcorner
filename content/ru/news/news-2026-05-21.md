### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.49
- Добавлена команда `/chronicle search` для поиска по истории сессий по ключевым словам и темам.
- Добавлена команда `/rubber-duck` для независимой критики текущей работы агента.
- Добавлена команда `/memory on|off|show` для управления постоянной памятью.
- Добавлены экспериментальные `/mcp search` и отложенная загрузка инструментов для MCP и внешних инструментов.

### GitHub Copilot CLI - 1.0.51
- Добавлена экспериментальная команда `/security-review` для проверки изменений кода на уязвимости безопасности.
- Добавлена команда `/chronicle cost-tips` с персональными рекомендациями по расходу токенов и снижению затрат.
- `/remote` теперь можно использовать, пока агент продолжает работу, и команда стала учитывать политики организации для удалённого управления.
- Добавлена настраиваемая строка состояния в терминале с данными о сессии, включая модель, окно контекста и ветку git.

### Visual Studio Code - 1.121
- Добавлена предварительная поддержка запуска и управления агентными сессиями на удалённых машинах из окна Agents.
- Добавлена наблюдаемость через OpenTelemetry и Grafana для операций агентов, расхода токенов, вызовов инструментов и задержек.
- Claude Agent получил предварительный режим Auto permissions для длинных задач с фоновыми проверками безопасности.
- VS Code теперь помечает команды терминала, запущенные агентом, переменной `VSCODE_AGENT` и улучшает работу фоновых терминалов в агентных сценариях.
