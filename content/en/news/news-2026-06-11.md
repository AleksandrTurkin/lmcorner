### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.57
- In Azure DevOps-only repositories, the built-in GitHub MCP server now exposes the `web_search` tool instead of being disabled entirely.
- `preToolUse` hook errors now block the tool call instead of silently allowing it, and MCP server timeout settings are preserved when tool lists change.
- Session recovery and prompt handling improved with better resume-after-crash behavior, quoted-path handling for `/skills`, and clearer auth rate-limit diagnostics.

### GitHub Copilot CLI - 1.0.58
- Rubber Duck is now enabled by default.
- Remote JSON RPC is now enabled by default.
- Experimental scheduled prompts with `/every` and `/after` are now available under `/experimental`, alongside a new GitHub UI for issues, pull requests, and gists.

### GitHub Copilot CLI - 1.0.59
- Added `/voice` to dictate prompts with local speech-to-text models.

### GitHub Copilot CLI - 1.0.60
- Added the maximum reasoning effort level for Anthropic models and made all effort levels available on every plan.
- Added `builtInAgents.rubberDuckAutoInvoke` to control automatic rubber-duck invocation.
- MCP tools added or removed by a server mid-turn now become available immediately, and BYOK file attachments larger than 5 MiB now work with the OpenAI Responses provider.
- Custom agent instructions are no longer duplicated each turn, reducing context-window usage.

### GitHub Copilot CLI - 1.0.61
- Added a polished `/agents` picker and Create New Agent wizard.
- MCP servers now auto-load from `.github/mcp.json`, and `/after` plus `/every` are listed in the experimental slash-command picker.
- Added support for Claude Fable 5 and improved Gemini compatibility with MCP tools that use nullable schema types.
- Added mTLS/private-CA support and standard OTLP HTTP/protobuf export options for telemetry.

### Visual Studio Code - 1.123
- Chat sessions can now sync to your GitHub account automatically, and `/chronicle` adds natural-language search across your coding history, standups, and productivity insights.
- The preview Agents window now supports multiple open sessions side by side for comparing and reviewing agent work in parallel.
- VS Code adds a preview research agent for deep, cited investigations in Copilot CLI Insiders sessions.
- Compatible Anthropic and OpenAI models now support 1M-token context windows.

### Visual Studio Code - 1.124
- Autopilot is now enabled by default, and Advanced Autopilot can use a small utility model to decide whether an agent task is actually finished.
- New background send support lets you launch a new agent session with Alt+Enter and keep composing the next one immediately.
- Keyboard-first session navigation and automatic layout restore make it faster to move across many agent sessions and reopen them after reloads.
- Browser history and the new `typeInPage` submit flow reduce round-trips for browser-driven agent tasks, and enterprise plugin policies now reuse the same Copilot policy file as Copilot CLI.

### Visual Studio 2022 - 17.14.34
- Copilot Agent Mode is now out of preview in Visual Studio 2022 and can use MCP-backed context from systems like GitHub, CI pipelines, and monitoring dashboards.
- Recent Agent Mode improvements add clearer progress indicators, prompt caching, working-set cleanup, a live Markdown planning tool, and manual control over automatic builds.
- Visual Studio 2022 adds GPT-5 support in Copilot Chat and general availability for MCP in Visual Studio, including broader OAuth support and easier MCP server setup.
- You can now connect your own Anthropic, Google, or OpenAI models to chat, partially accept completions, and hide next edit suggestions until you choose to review them.
- Visual Studio 2022 also adds new chat model options including Gemini 2.5 Pro, GPT-4.1, Claude Sonnet 4, and Claude Opus 4.

### Visual Studio 2026 - June Update 18.7.0
- The Copilot Usage window now shows real-time usage-based billing metrics, proactive alerts near limits, and configurable warning thresholds.
- Long-distance next edit suggestions let GitHub Copilot suggest related edits anywhere in the current file instead of only near the cursor.
- MCP server trust validation now checks both server configuration and loaded assets, prompting for re-approval when something changes.
- Pull requests can now be added directly to Copilot Chat, and the C++ GitHub Copilot modernization agent's MSVC upgrade scenarios are now generally available.
