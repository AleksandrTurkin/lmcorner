### GitHub Copilot CLI - 1.0.49
- Added `/chronicle search` to search session history by keyword or topic.
- Added `/rubber-duck` for an independent critique of the agent's current work.
- Added `/memory on|off|show` to control persistent memory.
- Added experimental `/mcp search` and deferred tool loading for MCP and external tools.

### GitHub Copilot CLI - 1.0.51
- Added experimental `/security-review` to review code changes for security vulnerabilities.
- Added `/chronicle cost-tips` for personalized token-usage and cost-reduction guidance.
- `/remote` can now be used while the agent is working and now respects organization remote-control policies.
- Added a customizable terminal status line showing session details such as model, context window, and git branch.

### Visual Studio Code - 1.121
- Added preview support for running and managing agent sessions on remote machines from the Agents window.
- Added OpenTelemetry and Grafana-based observability for agent operations, token usage, tool calls, and latency.
- Claude Agent now has an Auto permission mode preview for longer-running tasks with background safety checks.
- VS Code now marks agent-launched terminal commands with `VSCODE_AGENT` and improves background terminal handling for agent workflows.
