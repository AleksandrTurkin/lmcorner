### GitHub Copilot CLI - 1.0.56
- Пользователи Free и Student теперь могут выбирать в переключателе моделей не только режим Auto.
- Переключатель моделей теперь показывает корректный общий размер контекстного окна для каждого ценового уровня.
- Добавлен параметр `builtInAgents.rubberDuck`, позволяющий включать и отключать агент rubber duck через `copilot config`.
- Конфигурация BYOK-провайдеров теперь корректно применяется к ACP-сеансам.
- Инструменты MCP теперь передают агенту и человекочитаемый `content`, и `structuredContent`, не теряя ни одну из частей; дублирующийся JSON-текст автоматически удаляется.
- Переключатель reasoning effort теперь скрывает варианты, которые не поддерживаются выбранной моделью.
- Агент code review теперь использует ту же модель, что и текущий сеанс, вместо фиксированной модели по умолчанию.
- Выбор уровня контекстного окна теперь сохраняется между событиями сеанса и при возобновлении, поэтому лимиты запросов, сжатия и обрезки остаются согласованными с выбранным уровнем.
- Инструмент `web_fetch` теперь предпочитает Markdown, когда источник поддерживает согласование контента, что дает агенту более чистый контекст из документации.
