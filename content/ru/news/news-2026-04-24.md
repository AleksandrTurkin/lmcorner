### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.35
- Добавлена табуляция с автодополнением для slash-команд, их аргументов и подкоманд.
- Улучшено управление сессиями: появились именованные сессии, команды удаления, более умное возобновление через `--continue` и расширенный селектор сессий с ветками и статусами.
- Расширены агентные сценарии: появились самокорректирующиеся вызовы пользовательских инструментов, HTTP hooks, улучшения плагинов и LSP, а также shell access на Windows для configure agent.

### Visual Studio Code - 1.117
- Для Copilot Business и Enterprise появился BYOK: теперь можно подключать собственные API-ключи моделей прямо в чате VS Code.
- Улучшены чат и агентный опыт: добавлены инкрементальная отрисовка ответов, сортировка agent sessions и системные уведомления о фоновых terminal-командах.
- Развивается приложение Visual Studio Code Agents, а terminal получил более удобный запуск Copilot CLI и отдельные заголовки для agent CLI.

### Visual Studio 2022 - April Update 17.14
- Режим агента Copilot вышел из preview и получил более понятные индикаторы прогресса, улучшенное планирование, возможность отключать автоматические сборки и исправления зависаний.
- В чат Visual Studio добавлена поддержка GPT-5.
- Поддержка MCP стала общедоступной: появились расширенная OAuth-аутентификация, более простой способ добавления MCP-серверов и политики управления.
- В чат добавили подключение собственных моделей, частичное принятие автодополнений Copilot и скрытие Next Edit Suggestions по умолчанию.

### Visual Studio 2026 - April Update 18.5
- Агенты Copilot теперь автоматически обнаруживают и используют skills из репозитория или профиля пользователя.
- Из Visual Studio можно запускать cloud agent sessions на базе Copilot coding agent прямо из чата.
- Добавлены custom agents, настраиваемые горячие клавиши для встроенных подсказок Copilot, приоритет IntelliSense над Copilot completions и отдельная панель истории чата.
- Copilot также получил авторасшифровку текста в Text Visualizer и agentic issue-to-resolution workflow для отладки.
