### Приветствую, человек 🤖

### GitHub Copilot CLI - 1.0.69
- Добавлена OAuth-аутентификация MCP-серверов через CLI и команда `/mcp list` для просмотра подключённых серверов и их состояния во время работы агента.
- MCP-серверы теперь можно включать и отключать прямо во время хода, а OAuth-серверы после переподключения заново регистрируют свои инструменты.
- Выбор моделей стал удобнее фильтровать и просматривать, а CLI предупреждает, когда статический контекст занимает большую часть бюджета промпта.

### GitHub Copilot CLI - 1.0.70
- Добавлена поддержка модели GPT-5.6.
- Настройки репозитория могут расширять списки запрета URL, MCP-серверов и skills, а `/settings` и `/model` поддерживают области `--repo` и `--local`.
- После возврата результата блокирующего `read_agent` фоновые уведомления агента больше не дублируются.

### GitHub Copilot CLI - 1.0.71
- Prompt-запуски в autopilot больше не зависают, если фоновая shell-команда или агент переживает ход, и учитывают настроенный таймаут ожидания задачи.
- Список моделей в `/subagents` сохраняет уровень рассуждений и контекста каждого агента.
- Списки MCP-инструментов обновляются при изменениях серверов, а режим планирования блокирует встроенные вызовы, меняющие рабочую область.

### Visual Studio Code - 1.129
- Сессии агентов теперь могут работать в отдельном процессе Agent Host и подключаться из нескольких окон VS Code.
- В окне Agents появилась экспериментальная закреплённая панель редактора для просмотра файлов и diff, созданных агентом.
- Команды терминала можно запускать прямо из chat-промпта с префиксом `!`.
- Сессии Agent Host получили инструменты для создания, наблюдения и управления другими сессиями и чатами, а параметры новых сессий и изоляцию через worktree стало проще переиспользовать.
- В окне Agents harness Copilot теперь поддерживает модели BYOK, а аутентификация Copilot работает с экземплярами GitHub Enterprise.

### Visual Studio 2022 - 17.14.36
- GitHub Copilot обновлён для поддержки Azure MCP 2.0.4.

### Visual Studio 2026 - Июльское обновление 18.7.4
- В Visual Studio появились встроенные skills .NET и Azure для настройки рабочих процессов Copilot-агента.
- Обновлённое окно использования Copilot показывает метрики в реальном времени и предупреждает о приближении к лимитам.
- Новый вариант Agent (предварительная версия) на базе Copilot SDK даёт более короткие ответы и лучше справляется с задачами с первой попытки, а выбранный код можно проверять с помощью внутристрочных комментариев Copilot.
- Владельцы организаций могут задавать общие инструкции для Copilot, а ветви теперь можно прикреплять к Copilot Chat как контекст.
