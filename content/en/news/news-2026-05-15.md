### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.45
- Added `/autopilot` slash command to toggle between interactive and autopilot modes.
- OpenTelemetry output aligns with GenAI semantic conventions (new gen_ai.client.operation.duration metric for tool execution).

### GitHub Copilot CLI - 1.0.46
- Warns when the CLI version is deprecated and premium model access may be lost.

### GitHub Copilot CLI - 1.0.47
- Copilot Max subscribers now see the correct models available to their subscription tier.
- `--resume` supports Copilot cloud agent sessions where the agent hasn't pushed any changes to its branch.

### GitHub Copilot CLI - 1.0.48
- Model picker displays actual token prices for token-based billing users.
- `/context` shows correct token limits for all models.
- ACP clients receive updated config options when the active model is changed.
- `/ask` dialog no longer prompts for follow-up replies it cannot receive.
- Skill content injected to the model no longer includes YAML frontmatter metadata.

### Visual Studio Code - 1.120
- Agents window moved to Stable, providing an Agents workspace for orchestrating agent tasks across projects.
- BYOK improvements: accurate BYOK model token usage display and the ability to configure thinking effort for BYOK reasoning models.
- Risk assessment for terminal commands now shows an AI-generated explanation and risk badge.
- Terminal tool output compression reduces context usage when sending large terminal outputs to models.

### Visual Studio 2026 - 18.6.0
- View and manage Agent Skills from the Copilot chat window (skills panel for browsing, editing, and locating skills).
- Multi-file summary diff for Copilot changes: review and accept or undo edits across multiple files in one view.
- Context window indicator showing exact context usage with a one-click summarize control.
- Planning mode to draft implementation plans before making code changes, savable as markdown plans.
