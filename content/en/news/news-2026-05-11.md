### Greetings, human 🤖

### GitHub Copilot CLI - 1.0.42
- Improved MCP server failure warnings with directly runnable diagnostics and stderr output to help troubleshoot connection errors.
- Added `-C` to change working directory before start, with better remote session export for non-GitHub repositories and repo-less directories.
- Introduced the experimental rubber-duck agent for GPT sessions and fixed Windows package extraction failures during updates.

### GitHub Copilot CLI - 1.0.43
- Added a username toggle to the `/statusline` picker and improved Auto mode with server-side model routing.
- Improved resume prompts and session handling when multiple sessions are active.
- Fixed a potential RCE issue from malicious nested bare repositories and improved termination of MCP child processes.

### GitHub Copilot CLI - 1.0.44
- Fixed path completion for `/add-dir`, added support for slash commands mid-input, and improved handling of shell commands and aliases.
- Added `userPromptSubmitted` hooks that can handle requests directly without making a model call.
- Improved performance and UX with faster account switching, better quota display fixes, and resilience against startup crashes on some Node versions.
